## 未决 / Deferred

- **系列级一致性**（多期之间）：同构 hierarchy 递归套用，撞到痛点再实现。
- **shot 内 sub-part 拆分**（脚本 / 音频 / 画面 / 字幕）：等"重做 shot 但只想改 1 个 sub-part"出现 3 次以上再拆。
- **节点彻底 agent 化**（Phase 4）/ **整体 agent**（Phase 5）：前者按 eval 数据驱动，后者大概率永不需要。
