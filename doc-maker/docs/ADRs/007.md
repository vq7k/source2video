# ADR-007 · 编排框架：第一版不上 framework

**Status**: Accepted

**Decision**
- 5–8 节点：纯 Python + asyncio，**第一版禁止上编排 framework**。
- 状态复杂 / 节点间反馈多：LangGraph（显式状态图）。
- 想程序化优化 prompt：DSPy。
- 生产可靠性 / 长任务：Temporal / Inngest。
- 观测：Langfuse 或 Phoenix **必须早接**。

**Rationale**
- 早上 framework 会被它的抽象绑住，看不清节点本身的问题。
- 真正卡在"状态管理 / 重试"时再引入，触发器明确。

**Consequences**
- 第一版编排预期 < 300 行 Python。
- Langfuse 是唯一例外，第一版就要接。
