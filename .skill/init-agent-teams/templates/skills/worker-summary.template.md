# skill: worker-summary

Worker 完成任务后回主理角色的约定。

## 必写文件
1. `.agent/STATUS.md`（按 status-update）
2. `.agent/sessions/<YYYY-MM-DD>-<topic>/`：`context.md` / `decisions.md` / `outputs.md`
3. 踩坑后追加 `.agent/learned-rules.md`（无坑跳过）

## 口头 summary（≤5 行）
- 完成：<一句>
- 文件：<改了哪些>
- 阻塞：<无 / 列出>
- 待决策：<无 / 列出>
- 下一步：<建议>
